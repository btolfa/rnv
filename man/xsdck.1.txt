XSDCK(1)
========
:doctype: manpage

NAME
----
xsdck - example rnv plugin for XSD datatypes

SYNOPSIS
--------
*rnv -d xsdck* ['RNV_OPTIONS'] 'grammar.rnc' {document.xml ...}

DESCRIPTION
-----------
Both *RNV* and *RVP* can use pluggable datatypes. This module is an example providing support for XSD datatypes.
